신규 대체기기 발송 건 등록하기

사이드바의 '등록' 버튼을 클릭합니다.

사이드바의 등록 버튼 위치

등록 모달창이 열리면 다음 정보를 입력합니다.

  • 아이디: 학습생의 아이디
  • 이름: 학습생의 이름
  • 접수자: 발송 건을 접수한 담당자 이름
  • 등록일: 자동으로 오늘 날짜가 입력됨
  • 서비스: 학습생의 학년에 맞는 서비스 선택
  • 요청내용: AS대행인지, 대체기기 발송인지 등 요청 목적에 맞게 선택
  • 기종: S6 Lite와 A6 중 선택

등록 모달창(값 입력 전)

등록 모달창(값 입력 후)

'등록하기' 버튼을 클릭하여 정보를 저장합니다.

정상 등록 완료

등록을 완료하면 자동으로 대체기기 목록으로 이동됩니다.

해당 화면에서는 금일 접수된 기기 발송건만 보여지도록 필터가 적용되어 있습니다.

학습생의 아이디를 클릭하면 수정 모달창이 열립니다.

등록 후 자동 이동된 대체기기 목록

기존 정보 수정하기

대체기기 목록 또는 대시보드에서 학습생 아이디를 클릭하면 수정 모달창이 열립니다.

등록시 입력한 정보는 자동 노출되며, 배송 및 회수 일자는 직접 선택해야 합니다.

수정 모달창(상세 점검결과 옵션 비활성화)

점검결과 항목을 '무상' 또는 '유상'으로 선택하면 상세 점검결과에 옵션이 추가됩니다.

각 점검결과 옵션의 의미는 다음과 같습니다.

  • 없음: AS대상기 회수가 진행되지 않아 점검을 진행하지 않은 경우
  • 정상: 정상 판정으로 수리가 진행되지 않은 경우
  • 무상: 수리가 필요하지만 무상으로 진행되는 경우
  • 유상: 유상 수리 판정으로 고객에게 안내가 필요한 경우

수정 모달창(상세 점검결과 옵션 활성화)

필요한 값을 모두 입력하면 '수정하기' 버튼을 클릭하여 저장합니다.

수정하기 버튼 클릭시

오접수 등의 이유로 대체기기 발송 이력 자체를 삭제하려면 '삭제'버튼을 클릭합니다.

삭제할 경우 해당 건은 더 이상 확인할 수 없으며, 복구도 불가능합니다.
