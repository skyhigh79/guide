대체기기 목록 상세 구성

1. 주요 필터

사용자가 자주 사용하는 주요 필터 버튼입니다.

  • 전체: '종료 상태 제외' 필터도 제거되어 모든 대체기기 발송건을 확인할 수 있습니다.
  • 7일+ 지연건: 현재 진행단계에서 7일 이상 경과된 건들만 표시됩니다.
    대시보드 > 모니터링 필요 지연건 > 발송건 목록 모달창 > 대체기기 목록으로 이동 버튼을 클릭하였을 때 적용되는 필터와 동일합니다.
  • 내 접수건: 로그인한 사용자가 발송한 건들이 표시됩니다.

전체 필터

2. 현재 목록 엑셀 다운로드

'현재 목록 엑셀 다운로드' 버튼을 클릭하면 대체기기 목록의 내역을 Excel 파일로 다운로드할 수 있습니다.

현재 목록 엑셀 다운로드 버튼

다운로드된 파일명

* 다운로드된 파일을 실행한 후 '예'를 클릭하면 내용을 확인할 수 있습니다.

* Excel 파일에는 현재 적용된 필터에 해당하는 건들이 포함됩니다.

Excel 파일 실행 시 경고

Excel 파일 내용

3. 상세 필터

원하는 조건 및 기간에 해당하는 발송건을 확인할 수 있는 상세 필터 버튼입니다.

필터는 드롭다운 형식으로 되어 있으며, 여러 필터를 중복으로 적용할 수 있습니다.

필터를 중복으로 적용할 경우, 적용된 필터에 모두 해당하는 공통 발송건만 표시됩니다.

필터 중복 적용

같은 필터에서도 여러 옵션을 중복으로 선택할 수 있습니다.

필터가 적용된 상태에서 다른 옵션을 추가로 선택하면, 기존 옵션과 추가 선택한 옵션에 해당하는 건이 함께 표시됩니다.

옵션을 중복으로 적용할 경우, 적용된 옵션에 해당하는 건들이 모두 표시됩니다.

동일 필터 내 옵션 중복 적용

4. 검색

검색 버튼을 통하여 입력한 키워드가 포함된 발송건을 조회할 수 있습니다.

아이디, 접수자, S/N, 특이사항에 검색한 키워드가 포함된 발송건이 표시됩니다.

검색 기능

5. 상세 목록

상세 목록

  • No: 발송건을 등록하는 순서대로 번호가 부여됩니다. 기본적으로 오래된 건들이 위쪽에 노출되며, No 부분을 터치하면 최신 건들이 위쪽에 오도록 정렬됩니다.
  • 서비스: 초등, 중고등, 아이 중 해당하는 서비스가 표시됩니다.
  • 등록일: 발송건을 등록한 날짜입니다.
  • 아이디: 학습생의 아이디입니다.
  • 요청 내용: AS대행인지, 테스트기기만 발송하였는지 알 수 있습니다.
  • 접수자: CRM에서 실제로 대체기기를 발송한 접수자가 표시됩니다.
  • S/N: 기기의 시리얼 넘버와 기종이 표시됩니다.

  • 경과일: 현재 상태가 시작된 후 오늘까지 경과한 일수입니다. 경과일에 따라 행 색상이 다르게 표시됩니다.
    • 0-3일: 흰색
    • 4-6일: 노란색
    • 7일 이상: 빨간색
  • 점검결과: 회수된 고객 기기의 점검 결과입니다. (없음, 정상, 무상, 유상)
  • 특이사항: 발송건에 대한 추가 메모나 유의사항이 표시됩니다.
