대시보드 상세 구성

최근 60일 대체기기 발송현황

60일 이내 발송된 건을 카드별로 확인할 수 있으며, 발송 후 60일이 지난 건은 노출되지 않습니다.

수정 모달창에 입력된 값에 해당하는 카드에 발송건이 표시됩니다.

각 카드의 의미는 다음과 같습니다.

  • 전체 발송건: 최근 60일간 발송된 모든 대체기기 발송건
  • AS 진행 중: 대체기기 수령일이 값이 입력되었지만, 대체기기 회수접수일은 입력되지 않은 건
  • 회수 필요: 대체기기 회수접수일이 입력되었지만, 대체기기 회수완료일은 입력되지 않은 건
  • 회수 완료: 대체기기 회수완료일이 입력되어 종료된 건
  • 30일 경과 확인 필요건: 등록일로부터 30일이 경과하였지만 아직 회수 완료되지 않은 건

최근 60일 대체기기 발송현황

* 각 카드를 클릭하면 해당하는 발송건의 목록이 표시됩니다.

* 목록에서 학습생 ID를 클릭하면 수정 모달창이 열립니다.

카드 클릭 시 표시되는 발송건 목록

현황판 전체 내역 다운로드

'현황판 전체 내역 다운로드' 버튼을 클릭하면 대시보드의 내역을 Excel 파일로 다운로드할 수 있습니다.

현황판 전체 내역 다운로드 버튼

다운로드된 파일명

* 다운로드된 파일을 실행한 후 '예'를 클릭하면 내용을 확인할 수 있습니다.

* Excel 파일에 포함되는 내용은 최근 60일 대체기기 발송현황, 모니터링 필요 지연건, 소속 부서 발송건입니다.

모니터링 필요 지연건

처리가 지연되어 확인이 필요한 발송건이 표시됩니다.

현재 진행단계에서 7일 이상 경과된 건들이 해당됩니다.

+ 버튼을 클릭하면 해당하는 발송건의 목록이 표시되며, 학습생 ID를 클릭하면 수정 모달창이 열립니다.

모니터링 필요 지연건

* 발송건 목록 모달창 우측 상단의 대체기기 목록으로 이동 버튼을 클릭하면 대체기기 목록으로 이동합니다.

* 이 때 7일 이상 경과된 건들만 표시되도록 자동으로 필터가 적용됩니다.

모니터링 필요 지연건 모달창

월별 발송 건수

현재 월을 기준으로 최근 3개월간의 대체기기 발송 현황을 보여주는 그래프입니다.

최근 3개월과 작년 동일기간 3개월의 발송 건수를 비교할 수 있으며, 매달 1일이 되면 날짜에 맞게 자동으로 갱신됩니다.

소속 부서 발송건

로그인된 사용자가 소속된 부서에서 발송한 건들을 확인할 수 있습니다.

+ 버튼을 클릭하면 해당하는 발송건의 목록이 표시되며, 학습생 ID를 클릭하면 수정 모달창이 열립니다.
